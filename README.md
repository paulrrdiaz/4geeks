# Server

```
# Go to server folder
cd server

# Create virtual environment
python -m venv .venv

# Activate the virtual environment
source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run project in debug mode
flask run --debug
```

# Client

```
# Go to client folder
cd client

# Install dependencies
npm install

# Run project
npm run dev
```
